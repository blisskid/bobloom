== Description ==

Headstart is a responsive blogging and corporate agency WordPress theme. It's main purpose is to be used as a blog, but the widgets allow you to create a business landing page or product marketing page for an app for example. Headstart is build on Foundation framework, it is fully responsive so it works on both mobile and desktop on Android, iPhone and all other devices. The design is flat and simple, so it works for writers and bloggers as well, it's optimized so the typography is easy to read, making it ideal for a magazine / newspaper website as well. The widgets can also be used for affiliate ads such as adsense or anything else you'd like. We have had a lot of focus on search engine optimization, so it's fully SEO friendly and allows you to rank good in Google, Bing, Yahoo and all other search engines. The elegant design is good for creating a personal writing portfolio about lifestyle, fashion or anything else. It's easy to customize, you can add header images and change header colors. We will keep updating the theme with better SEO optimization when Google change their algoithm so you can keep ranking well.
